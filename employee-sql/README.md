Este repositorio contiene una serie de ejercicios prácticos de SQL 
basados en un dataset de empleados.

El objetivo es practicar consultas SQL en PostgreSQL y reforzar los 
conocimientos básicos de SQL, aplicándolos a un contexto más real 
de análisis de datos y toma de decisiones empresariales.

El dataset utilizado proviene de Kaggle:
https://www.kaggle.com/datasets/tawfikelmetwally/employee-dataset
